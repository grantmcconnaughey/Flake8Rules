---
code: E242
message: "Tab after ','"
title: "Tab after ',' (E242)"
links:
  - https://www.python.org/dev/peps/pep-0008/#pet-peeves
---

There should be only one space after the `,` character.

### Anti-pattern

**Note:** The `→` character represents a tab.

```python
my_tuple = 1,→2,→3
```

### Best practice

```python
my_tuple = 1, 2, 3
```
