---
code: E302
message: "Expected 2 blank lines, found 0"
title: "Expected 2 blank lines, found 0 (E302)"
links:
  - https://www.python.org/dev/peps/pep-0008/#blank-lines
---

Two blank lines are expected between functions and classes.

### Anti-pattern

```python
def func1():
    pass
def func2():
    pass
```

### Best practice

```python
def func1():
    pass


def func2():
    pass
```
